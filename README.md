# CNC Mill Hub

CNC milling reference site with calculators, skills content, and tools. Mobile-friendly, dark mode, modern UI.

## Features

- **Calculators**: Feed rate, spindle speed (RPM), chip load, material removal rate, cutting time
- **Skills**: CNC milling basics, feeds/speeds, tool selection, workholding, G-code, safety
- **Tools**: Unit converter (inch/mm), fraction↔decimal, tap drill chart, SFM reference

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
